About the Project

This system uses a trained machine learning model to predict the likelihood of breast cancer based on user input. It aims to assist in early detection and empower users with knowledge.

  • Built using Flask (Python backend) and Bootstrap (frontend)
  • Trained on the UCI Breast Cancer Wisconsin Dataset
  • Model: Random Forest Classifier with 95%+ accuracy
  • User-friendly web interface for real-time prediction

Explore More

Why Early Detection Matters

Early detection improves treatment success and increases survival rates. Our model helps raise awareness for timely diagnosis.

Technologies Used

The system leverages Python, Flask, HTML5, Bootstrap 5, and Scikit-learn for building a scalable, reliable, and responsive prediction tool.

About the Dataset

The UCI Breast Cancer dataset contains key diagnostic features such as radius, texture, perimeter, and smoothness used in prediction.
